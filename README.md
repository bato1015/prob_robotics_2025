## 確率ロボティクスの課題提出場所
  ### 変分推論を用いたノイズ除去システム
  Microsoft社製のAzure Kinect DK[1]で計測した点群データを使用し，変分推論を用いたノイズ除去システムを構築した．
  1視点からの計測でオクルージョンが多く発生することから奥行き方向であるy軸を除いたx,z方向の2次元に変換してから行った．
